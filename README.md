# 押印 — 推し活観測ログ

推しメンバーごとに観測記録を積み上げるダークモード専用 Web アプリ。

**デモ:** https://oshi-log-shizuku.vercel.app

---

## 作った目的

「何を、いつ、どんな感情で記録したか」を静かに積み上げていける場所がほしかった。
既存の SNS や手帳アプリでは情報が流れてしまうため、手元に残るログブック形式で実装した。

## 工夫した点

- **CSS カスタムプロパティによるトークン設計** — メンバーごとに `--accent-color` が動的に切り替わり、配色が変わる仕組みを実装
- **Vanilla JS + localStorage のみ** — 外部依存ゼロ。オフライン環境でも動作する軽量構成
- **ダークモード専用 UI** — 目に優しい配色と余白設計で、夜の記録に特化したレイアウト
