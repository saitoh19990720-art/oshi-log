# 押印 — 推し活観測ログ

推しごとの観測記録を静かに積み上げるログブック。ダークモード専用デザインで、推しメンバーごとにタイトル・日付・メモを記録できる。

**公開:** （Vercel にデプロイ後にURLを記載）

## 作った目的

日々の推し活で「見た・聞いた・感じた」を忘れないうちに残す場所が欲しかった。SNSだと流れてしまい、ノートアプリだと情報が混ざる。推し専用に切り出して、静かに積み上げるだけの道具にした。

## 工夫した点

- **ダークモード専用の UI 設計**: 夜に開くことが多い前提で、長時間見ても疲れないトーンに統一。グラス化パネルと `--accent-color` CSS変数でメンバーごとに色味が変わる
- **localStorage のみで完結**: バックエンド無しでデプロイ1クリック。オフラインでも動く
- **サイドバー切替 → 即記録**: 画面遷移を減らして、思い立った瞬間に1件追加できる導線

## 使った技術

HTML / CSS / JavaScript（Vanilla） / localStorage / Vercel（静的ホスティング）

## ローカル実行

```bash
# ルートで index.html を開くだけ
# もしくは
npx serve .
```

## ファイル構成

- [index.html](index.html) — レイアウト（サイドバー + ワークスペース）
- [styles.css](styles.css) — ダークテーマ / トークン設計 / グラス化UI
- [app.js](app.js) — 状態管理 / フォーム処理 / localStorage 永続化
