# Shizuku | AI x Design

Figma、AI、コードを使って、小さなWebページや制作補助ツールを整えるための静的サイトです。

## 内容

- LP / 小さなWebページの制作メモ
- UI整理の考え方
- AI制作環境の分業ルール
- 推し活観測ログなどの軽量Webツール

## 技術

| 項目 | 内容 |
|---|---|
| 画面 | HTML / CSS / JavaScript |
| 保存 | localStorage |
| デプロイ | Vercel想定 |
| 依存 | なし |
