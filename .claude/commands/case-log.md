案件の状態管理ログ。応募・返信・受注・納品の記録を整形して出す。Obsidianへの貼り付け用。

## 使い方

状況（応募した / 返信きた / 受注した / 納品した）と案件名をそのまま伝える。

---

## 出力フォーマット

### 応募時

```
## [案件名]
- 状態: 応募済み
- 日付: YYYY-MM-DD
- タイプ: [A/B/C/D]
- 予算: [円 or 応相談]
- 注目点: [採用・落選に影響しそうな1点]
- 応募文: [提出した本文 or 「/proposalで生成」]
```

### 返信・受注時

```
## [案件名]
- 状態: [返信あり / 受注]
- 日付: YYYY-MM-DD
- 内容: [先方の要望・条件変化など1〜2行]
- 次アクション: [何をいつまでに]
```

### 納品時

```
## [案件名]
- 状態: 納品完了
- 日付: YYYY-MM-DD
- 納品物: [ファイル名 or URL]
- 実績化: [可 / 要確認 / 不可]
- 事例メモ: [1行・何を作って何が学びだったか]
```

---

## 制約

- 各フィールドは1〜2行以内
- 感想・評価は入れない
- 実績化可能な案件は「事例メモ」を必ず入れる
