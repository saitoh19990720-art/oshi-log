# LP確認チェック

## 構造
- [ ] ファーストビューで何の人かわかるか
- [ ] セクション順が自然か
- [ ] CTAまで無理なく読めるか

## 見た目
- [ ] 余白が詰まっていないか
- [ ] 水色が強すぎないか
- [ ] SaaSテンプレ感が強すぎないか
- [ ] 可愛すぎず、静かに整っているか

## スマホ
- [ ] 縦に読んで苦しくないか
- [ ] ボタンが押しやすいか
- [ ] 見出しが長すぎないか

## 文
- [ ] 営業っぽすぎないか
- [ ] 抽象語が多すぎないか
- [ ] 自分の仕事がちゃんと伝わるか

---

3つ以上❌ならまず修正プロンプト（`/lp-revise`）を使う。
