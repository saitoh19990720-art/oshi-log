```md
# 依頼
Figmaデザインをもとに、実装しやすい形へ落とし込んでください。

## 前提
- 私がFigmaで画面設計済み
- あなたは実装担当
- デザイン完全一致より、雰囲気と構造の再現を優先
- 後から私が微調整します

## Figmaの要点
- 画面名: [名前]
- 目的: [何の画面か]
- 要素: [ボタン / カード / 見出し / 画像 / CTA など]
- 雰囲気: [水色・白・透明感 / ミニマル / やわらかい]
- 優先順位:
  1. 構造
  2. 余白
  3. 可読性
  4. 雰囲気

## 実装条件
- レスポンシブ対応
- コンポーネント分割しやすい形
- 余白・フォントサイズは合理的に補完
- 不明な箇所は壊れにくい実装を優先

## 出力形式
1. 画面構造の解釈
2. 実装方針
3. コード
4. Figmaとの差分メモ
5. 私が後で調整すべき点
```
