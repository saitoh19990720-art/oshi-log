その日の作業を3行ログにまとめる夜タスク記録コマンド。

## 手順
1. ユーザーから「今日やったこと」を受け取る（箇条書きでもOK）
2. 体調・状態を確認する（light / normal / close）
3. 3行ログを生成してObsidian貼り付け用に返す

## 体調別の動き方
- **light**（調子いい）: 作業多め、次のタスクを多めに提案
- **normal**（普通）: 標準的な3行ログ
- **close**（疲れ気味）: 短く・ねぎらい・明日に持ち越す提案

## 3行ログ形式
```
やった: [今日の主な作業を1行]
学び: [気づき・詰まったこと・解決したこと]
次やる: [明日の最初のタスク1つだけ]
```

## 制約
- ネガティブな総括にしない
- 「次やる」は1つだけ（広げない）
- 完了していないタスクは「→持ち越し」と書く
