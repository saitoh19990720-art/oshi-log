# /proposal-short — 短い提案文テンプレ

CrowdWorks / Lancers / ココナラDM で、**最初の一言メッセージ**として送る短文。
長文版 `/proposal` を送る前のアイスブレイク、または案件内容が軽いときにそのまま使う。

## 基本テンプレ

```md
はじめまして。
Figmaでの設計からHTML/CSSでの実装まで一貫して対応しています。

LP制作では、
・ユーザーが迷わず読める構成
・スマホでも崩れにくい設計
・あとから修正しやすい実装
を意識しています。

まずは小さく形にしながら調整していく進め方が得意です。
もし方向性が合いそうでしたら、一度お話伺えればと思います。
```

## 世界観寄りバリアント（推し活・Z世代ブランド向け）

```md
はじめまして。
推し活や Z世代向けブランドのLPを、静かな世界観を崩さずに実装するのが得意です。

FigmaからHTML/CSSへ一気通貫でつなぎながら、
・読みやすさと導線を最優先に
・スマホでも崩れない構造
・装飾より余白と文字組みで整える
という進め方をしています。

世界観のすり合わせが重要な案件ほど相性が良いので、
方向性が合いそうでしたらメッセージでやり取りさせてください。
```

## 既存改善（リライト/修正）向けバリアント

```md
はじめまして。
既存LP/Webサイトを、世界観を保ったまま導線と可読性だけ整えるのが得意です。

最小差分で、情報設計・余白・CTA配置を中心に直します。
フルリニューアルではなく「静かに改善」する方針で、
元のブランドらしさは崩しません。

対象URLをいただければ、先に改善ポイントを3点ほどお返しします。
```

## 使い方

- 応募の最初の投稿はこれで十分（長文版だと警戒される場合あり）
- 相手が返信 or 要件詳細を送ってきたら `/proposal` でフル応募文に切替
- 価格提示はこのメッセージではしない（次のやり取りで出す）
- テンプレ感を消すため、相手の案件本文から**1行だけ引用して言及**すると反応率が上がる

## 注意

- 絵文字は使わない（案件応募では避ける）
- 「ぜひ」「喜んで」系は入れない（押しが強くなる）
- 分量は 15行以内に収める
