# Lancers 出品文②｜既存サイト改善

## パッケージ名
既存サイトの見づらさ・離脱ポイントを改善します

## キャッチコピー
「なんか見づらい」をHTML/CSSで具体的に直します。通話不要。

## サービス内容

稼働中のサイトの「見づらさ」「伝わらなさ」をHTML/CSSで改善します。
全面リニューアルではなく、効果の高い箇所を絞って最小限の修正で対応します。

**よくある改善依頼**
- ファーストビューで何のサービスかわからない
- スマホで崩れている・読みにくい
- ボタンが目立たない・クリックされない
- 情報が詰まりすぎて離脱している

**対応範囲**
- 現状の課題整理（3点以内）
- HTML / CSS修正（指定箇所）
- 改善後の確認URL発行
- 修正対応 2回まで

**対応しないこと**
- WordPress・CMSのテンプレート変更
- 全ページリニューアル（別途相談可）
- PHPなどサーバーサイドの改修

## 制作の流れ

1. サイトURLと「気になっている点」をご共有ください
2. 課題と対応方針をご提示してから着手します
3. 修正後に確認URL共有 → 完了

## 納期目安

軽微な修正（1〜3箇所）：1〜3営業日
