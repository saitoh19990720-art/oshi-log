# Lancers 出品文①｜Figma → LP実装

## パッケージ名
FigmaデザインをHTML/CSSでLP実装します

## キャッチコピー
デザインデータそのまま、崩さず実装。通話不要・テキスト完結。

## サービス内容

FigmaのデザインファイルをHTML/CSSでそのまま実装します。
世界観を崩さず、PCとスマートフォン両方で正しく表示できる状態で納品します。

**対応範囲**
- HTML / CSS コーディング（1ページ）
- レスポンシブ対応（PC 1280px・スマホ 375px）
- 確認用URL発行（Vercel）
- 修正対応 2回まで

**対応しないこと**
- デザイン制作（Figmaデータをご用意ください）
- WordPress・CMS組み込み
- 複雑なJavaScript・アニメーション（ご相談ください）

## 納品物

- HTML / CSS ファイル一式
- 確認用Vercel URL（ご希望に応じてファイル納品も可）

## 制作の流れ

1. Figmaデータをご共有ください（閲覧権限のみでOK）
2. 実装内容・納期をご確認のうえお見積もり
3. ご承認後、制作開始
4. 確認URL共有 → 修正 → 完了

やり取りはすべてテキストで完結します。通話は不要です。
