# Claude Codeへの指示テンプレ

よく使う依頼文の型。コピペして使う。

---

## 実装依頼

```
目的：
[何のページか]

技術：
HTML/CSS（最小構成）

条件：
- スマホ優先
- 最小構成で成立させる
- 崩れにくく
- 後から直しやすく

出力：
1. 実装方針
2. ファイル構成
3. HTML
4. CSS
5. 次に確認すべき点
```

---

## 修正依頼

```
方向性はOKです。
次だけ直してください。

- [修正点1]
- [修正点2]

全体を書き直さず、必要部分だけ直してください。
```

---

## Figma → 実装依頼

```
Figmaの意図を汲んで実装してください。
完全一致より、構造・余白・雰囲気の再現を優先してください。
後から私が微調整しやすい形にしてください。
```

---

## 運用ルール（毎回意識する）

- 1案件 = 1スレッド（`/clear` で切る）
- 1回 = 1目的（混ぜない）
- まず最小構成、後から拡張
- 修正は優先順位つきで投げる
- 完成したらテンプレ化
