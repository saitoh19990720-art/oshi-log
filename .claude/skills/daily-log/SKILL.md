# Daily Log Skill

## 使い方
「daily-log を使います」と言うだけ。体調と今夜やることを答えれば整形して返す。

## 夜タスクテンプレ

```
# 夜タスク｜{{date}}

体調：[ light / normal / close ]
外出や消耗：[ 多い / 少ない ]
明日を守る必要：[ ある / ない ]

## 仕事前ルーティン
22:50 Qiita 10分（点火として使う・探索しない）
23:00 今日やる1つを決める
23:03 作業開始

今夜のレーン：
- [ ] 日本案件
- [ ] 母艦サイト
- [ ] Claude固定運用
- [ ] 今日はクローズ

タスク：
最初の1手：
完成条件：

---
やった：
学び：
次やる（最初の1手）：
```

## 体調別の動き方

| 体調 | 動き |
|---|---|
| normal | 通常タスクを1つ進める |
| light | Qiitaだけ → クローズでもOK |
| close | 今日は休む。3行ログだけ残して終わり |

## 3行ログ（最小版）

```
やった：
学び：
次やる（最初の1手）：
```

## 制約
- タスクは1つだけ
- 完成条件は「公開できる状態」
- 増やす提案より固定化・圧縮を優先
- 今週の勝ち条件を確認してからレーンを選ぶ
