About

わたしについて

水色透明系の世界観を核に、推し活と美学を整える道具としてのWeb・アプリ・テンプレをつくっています。

コントラストは弱く、輪郭は柔らかく、光は拡散させて。ギラギラした加工ではなく「霧をかける」ような静かな補正が得意です。

FigmaとAIで世界観をそのまま形にする制作フローを組み、SNS前提のビジュアルやZ世代向けブランド、推し活系サービスに寄り添います。

Design Language

色と空気

aqua

水色（主軸）

#A8C4CC。透明感のあるベースカラー。全ての空気の起点。

dusty pink

くすみピンク

#D4A8B8。甘さを足しすぎない補助色。低彩度で静かに効かせる。

lavender

ラベンダー

#C4B4DC。儚さと余韻。影やアクセントに使うと世界観が立つ。

beige

ベージュ

#D8C8B4。生成りの地肌。真っ白より少し温度がある土台。

motifs

モチーフ

天使・羽・泡・ガラス・月・雫・リボン・鏡。硬質でなく透ける素材。

rules

レイヤーの作法

ベタ塗り1枚より半透明2〜3枚。角丸・細線・広い余白・1画面1感情。

Works

制作物

oshi-log

推し活ログアプリ

推しごとの記録・検索・ライト／ダーク切替。Yomogiの手書きで静かな可愛さを出したUI。

Stillwater

LPデモ

静謐と透明感をテーマにしたLPデモ。ラベンダーの儚さでトーンを整える。

Lumière

サロンLPデモ

水色 × くすみピンクの浮遊感。美容系を美学の道具として再設計したデモ。

banner

バナーサンプル

生成りの地肌に、水色・ラベンダー・ピンクを薄く重ねたSNS用バナー集。

Process

制作の流れ

01

世界観を決める

機能より先に「どんな存在を世に出したいか」を固定する。作品先行の起点。

02

色と空気を組む

水色を主軸に、くすみピンク・ラベンダー・ベージュで補助輪をかける。低彩度・拡散光。

03

Figmaで設計

カラートークン・タイポ・角丸・半透明レイヤーをコンポーネント化する。

04

AIで実装

Figma MCP × Claude Code で世界観を崩さず、公開できる状態まで整える。

05

静かに仕上げる

盛りすぎない。情報量が多くても見え方は静かに。非同期・テキスト中心で認識を揃える。
