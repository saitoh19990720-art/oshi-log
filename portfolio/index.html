About

わたしについて

ホビ垢文化が好きで、地雷／天使／水色界隈の"らしさ"を、FigmaとAIを使ってプロダクトやテンプレに落とし込んでいます。

界隈ごとの色・フォント・モチーフ・空気感をデザインシステムとして整理し、Figma MCP × Claude Code で「世界観をそのまま再現できる制作フロー」をつくっています。

SNS前提のビジュアル、Z世代向けブランド、推し活系サービスなど、"界隈が分かる人"に任せたい案件を歓迎します。

Design System

界隈別デザインシステム

landmine

地雷

黒・ピンク・赤／ハート・十字架・涙。太め丸文字＋病み英字で、甘さと不安定さを同居させる。

angel

天使

白・淡ピンク・シルバー／羽・リボン・星屑。細い手書き＋薄エフェクトで儚さと透明感を出す。

aqua

水色

水色・白・シルバー／泡・氷・透明。丸ゴ＋浮遊感のあるレイアウトで清潔と涼しさを作る。

Works

制作物

oshi-log

推し活ログアプリ

推しごとの記録・検索・ライト／ダーク切替。Yomogiフォントで手書き感を出した地雷寄りUI。

Stillwater

LPデモ

静謐・透明感をテーマにしたLPデモ。天使界隈寄りのトーン設計。

Lumière

サロンLPデモ

水色×シルバーの浮遊感。サロン・美容系を水色界隈の文脈で再設計したデモ。

banner

バナーサンプル

界隈テイストを織り込んだSNS用バナーのサンプル集。

Process

制作の流れ

01

界隈リサーチ

地雷／天使／水色のどの文脈に寄せるか、色・フォント・モチーフを言語化する。

02

Figmaで設計

界隈デザインシステムを元に、カラースタイル・タイポ・コンポーネントを構築。

03

AIで実装

Figma MCP × Claude Code で、世界観を崩さず公開できる状態まで実装する。

04

テキストで確認

非同期・テキスト中心で認識を揃えながら仕上げる。通話不要。
