# 導線文セット｜プロフィール・Hero文・応募テンプレ

---

## X プロフィール（本命）

```
AI×デザインでLPやWebページを作る人
Figma設計 × Claude Code実装
非同期 / UI設計 / AI×デザインワークフロー
```

---

## CrowdWorks プロフィール

```
AI×デザインで、LPやWebページを制作しています。

Figmaでの設計から、Claude Codeを活用した実装まで一気通貫で進めるのが強みです。非同期・テキスト中心でやり取りしながら、公開できる状態まで丁寧に整える進め方を重視しています。

対応しやすい内容
・LP / サービス紹介ページ制作
・Figmaでの画面設計
・HTML/CSSなどの実装
・構成整理 / 導線整理

「通話前提ではなく、文章で整理しながら進めたい」「小さくても形にしたい」という案件と相性が良いです。
```

## 応募テンプレ完成版

```
こんにちは。案件を拝見し、LP制作の目的や導線設計まで含めてお手伝いできると感じ、ご連絡しました。

私は、Figmaでの設計からClaude Codeを活用した実装まで、一気通貫で進めています。LPやWebページを、非同期・テキスト中心で整理しながら形にする進め方が得意です。

途中共有もテキストベースで行い、認識のズレを減らしながら進行します。納期や進め方も内容に合わせて調整できます。よろしくお願いいたします。
```

**差し替え3項目**
1. 案件名 / 業種（冒頭1文に入れる）
2. 近い実績や経験
3. 納期 / 技術スタック / 稼働条件

**短縮版**
```
案件を拝見し、LP制作の目的や導線整理まで含めて対応できると感じご連絡しました。Figma設計からClaude Codeを活用した実装まで一気通貫で進めています。非同期・テキスト中心で整理しながら形にできます。
```

## CrowdWorks応募コピペ順

```
1. 案件本文をコピー
2. proposal 入力テンプレに貼る
3. 関連実績・納期・強みを埋める
4. Claudeに「proposal を使います」と貼る
5. 完成版を見る
6. 差し替え3項目だけ調整
7. CrowdWorksに貼る
8. 送信前に冒頭だけ確認
```

**送信前の確認**
- 案件理解の一文がある
- 非同期・テキスト中心が自然に入っている
- 長すぎない・盛っていない
