## .claude/skills/proposal/SKILL.md

```markdown
---
name: proposal
description: 日本案件向けの応募文・提案文を高速生成する。CrowdWorksやココナラなどの案件文を読み、非同期・テキスト中心の進め方と、Figma設計からClaude Code実装までの一気通貫の強みを自然に含めた提案文を作る。
---

# proposal

## 目的
日本案件向けの応募文を毎回ゼロから考えずに安定して作る。

## 入力
- 案件URL または案件本文
- 関連実績・納期・強み・技術スタック

## 出力
1. 完成版
2. 差し替えポイント3項目
3. 短縮版（100〜200字）

## ルール
- 冒頭に案件を読んだ感じの1文を入れる
- Figma設計→Claude Code実装の一気通貫を自然に入れる
- 非同期・通話不要・テキスト中心を押しつけずに伝える
- 長すぎない・過度に盛らない・そのまま送れる文面
```

```markdown
## ルール
- 最小構成で始める・「公開できる状態」を優先
- ライブラリ追加は勝手に行わない
- 必要なときだけ質問する
```
