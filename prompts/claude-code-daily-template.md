# Claude Code｜Obsidian実行テンプレ

> Claude Codeには「目的・範囲・出力・停止条件」だけ渡す。

**作成日：** 2026-04-17
**格納先：** `prompts/claude-code-daily-template.md`

---

## 🎯 結論

Claude Code用も、Obsidianにそのまま貼れる **実行テンプレ** にすると使いやすい。

- 「目的・範囲・出力・停止条件」を固定 → 安定
- 夜タスクは毎回同じ枠でOK
- ポイントは **広げない・止める・次に残す**
- `CLAUDE.md` は恒常ルール／このテンプレは **その日の実行指示**
- Obsidianでは **Daily Noteから呼ぶ形** が相性いい
