# Purpose Memo

## このプロジェクトの現状

- Git は初期化済み
- 実装ファイルはまだない
- まだ用途を決める段階

## 用途候補 3案

### 1. 個人用タスク整理Web

- 毎日の作業を小さく管理できるWebツール
- 画面を作る練習と、状態管理の基本を学びやすい
- 最小構成で完成まで持っていきやすい

### 2. 作業メモ整理アプリ

- メモ、調査ログ、次の行動を整理するツール
- あとから検索やタグ付けに広げやすい
- アプリ化すると設計が少し広がる

### 3. 開発実験用プロジェクト

- UIや構成、各種ツールを自由に試すための場
- 学習には向くが、完成物の形がぼやけやすい
- 目的が散りやすいので、継続には工夫が必要

## 今の自分に一番向いている案

**個人用タスク整理Web**

### 選んだ理由

- 何を作るかを具体化しやすい
- 小さく始めて完成まで持っていきやすい
- Webの基本要素を一通り触れる
- 今後、必要ならメモ機能や記録機能も足せる

## 当面の方針

- まずは1画面だけを目標にする
- 保存は複雑なバックエンドなしで始める
- 完成優先で、小さな機能から積み上げる
